# Inova Mais

## Descrição do Projeto

Este repositório contém um sistema desenvolvido em **Python** para automatizar processos da Unidade Básica de Saúde (UBS), tornando o atendimento, o registro de informações e a gestão de dados mais ágeis e eficientes para os profissionais da saúde. O sistema é executado diretamente no terminal do **Visual Studio Code (VS Code)**.

## Funcionalidades Principais

- Automatização de tarefas administrativas e operacionais da UBS
- Registro e consulta de pacientes
- Agendamento de consultas e procedimentos
- Gerenciamento de prontuários e histórico de atendimentos
- Geração de relatórios para apoio à gestão

## Tecnologias Utilizadas

- Linguagem de programação: **Python**

## Como usar

Para executar o sistema localmente utilizando o **Visual Studio Code (VS Code)**, siga os seguintes passos:

1.  Certifique-se de ter o **Python** e o **pip** instalados em sua máquina.
2.  Abra o VS Code e clone este repositório utilizando a função "Clone Git Repository...".
3.  Abra a pasta do projeto no VS Code ("File" > "Open Folder...").
4.  Abra o terminal integrado do VS Code ("Terminal" > "New Terminal").
5.  No terminal, navegue até o diretório onde se encontra o arquivo principal do sistema (ex: `main.py` ou similar).
6.  Execute o sistema utilizando o comando: `python nome_do_arquivo_principal.py` (substitua `nome_do_arquivo_principal.py` pelo nome real do arquivo).

**Dica para VS Code:** Você pode configurar um ambiente de execução (Run and Debug) no VS Code para facilitar o debug e a execução do seu projeto Python diretamente no terminal. Consulte a documentação do VS Code para Python para mais detalhes.
